# Code of Conduct

## Our Pledge

We are committed to making VLC RPC a welcoming and respectful project for everyone.

## Expected Behavior

- Be respectful and constructive.
- Report bugs with enough detail to reproduce them.
- Keep discussions focused on improving the project.
- Respect maintainers' time and decisions.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Spam, trolling, or intentionally disruptive comments.
- Sharing private information without permission.

## Enforcement

Project maintainers may remove comments, close issues, block users, or take other reasonable action when behavior violates this code.

## Reporting

Report conduct concerns by opening a private contact channel with the project maintainer if available, or by using GitHub's reporting tools.
